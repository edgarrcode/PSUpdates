What is an Asset?

Answer:
An asset is an owned item of value. Items are determined to be assets according to physical properties and value.

What is an Asset Profile ID?

Answer:
The Asset Profile is an alphanumerical identifier used to code an asset.

Why should I code assets?

Answer:
Coding assets will allow for the proper tracking of the identified asset in PeopleSoft.

In what system do we enter the Asset Profile ID?

Answer:
In Miner Mall.

Are office supplies considered assets?

Answer:
No; consumable products are not considered assets.
